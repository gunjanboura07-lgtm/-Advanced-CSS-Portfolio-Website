# Advanced CSS Portfolio Website
## Project Overview
This project is a responsive and interactive portfolio website developed using HTML, CSS, and JavaScript. The website demonstrates advanced CSS concepts such as CSS Grid, Flexbox, CSS Variables, Animations, Responsive Design, and BEM Methodology.

## Features
- Responsive Design
- CSS Grid Layout
- Flexbox Layout
- Dark Mode Toggle
- Smooth Animations and Transitions
- CSS Variables
- Mobile-First Design
- BEM Naming Convention


## Technologies Used
- HTML5
- CSS3
- JavaScript

## Advanced CSS Techniques Used

### CSS Grid
Used for creating responsive layouts and project sections.

### Flexbox
Used for navigation bar, alignment, and form layout.

### CSS Variables
Used for managing colors and themes.

### Animations and Transitions
Used for smooth hover effects and interactive UI.

### BEM Methodology
Used for organized and reusable CSS structure.

## Responsive Design
The website is fully responsive and works properly on:
- Mobile Devices
- Tablets
- Desktop Screens

Media queries are used for responsiveness.


## How to Run the Project
1. Download or clone the project
